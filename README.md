# Distance Love Site

Простое React/Vite приложение для романтического сайта отношений на расстоянии.

## Запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите локально:

```bash
npm run dev
```

3. Откройте адрес из терминала, например `http://localhost:5173`.

## Что внутри

- приветственная секция
- счетчик дней вместе
- таймер до следующей встречи
- карта расстояния
- таймлайн воспоминаний
- личная заметка
